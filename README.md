# Mini Music

## Setup

pip install -r requirements.txt

## Run locally

uvicorn api.main:app --reload

## Deploy to Vercel

1. Install Vercel CLI
2. vercel deploy

## Set Webhook

https://api.telegram.org/bot<TOKEN>/setWebhook?url=YOUR_URL/webhook

## Features
- Telegram Bot (Webhook)
- Mini Music Player
- FastAPI backend
- Serverless ready
